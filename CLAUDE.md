## 🎯 서비스 개요

**서비스명:** 엔로그 (Enlog)
**도메인:** enlog.kr (사용 가능 확인 완료)
**한 줄 설명:** 한국 N잡러와 프리랜서를 위한 수입 관리 + 세금 계산 앱

### 우리가 푸는 문제
사업자 등록 없이 3.3% 원천징수로 일하는 N잡러들을 위한 서비스가 없다.
- 기존 앱(머니핀, 비즈넵)은 사업자 등록자 대상
- 해외 앱(Bonsai, HoneyBook)은 한국 세금 체계 미지원
- 55만 N잡러가 수입 관리를 카카오톡 + 엑셀로 하고 있음

### 타겟 사용자
- 20~35세, 사업자 등록 없이 프리랜서로 부수입 있는 직장인
- 디자이너, 개발자, 영상 에디터, 번역가, 강사
- 3.3% 원천징수로 수입을 받는 모든 N잡러

## 🚀 MVP 핵심 기능

1. **수입 기록** — 클라이언트, 금액, 날짜 입력 / 실수령액 자동 계산 (3.3% 공제)
2. **세금 계산** — 종합소득세 환급 예측 / 건강보험료 인상 경고 (168만원 초과 시)
3. **프로젝트 관리** — 클라이언트별 프로젝트 진행 상태 관리
4. **간편 계약서** — 한국식 프리랜서 계약서 템플릿 + PDF 생성

### MVP 제외 (v2 이후)
- AI 세금 상담
- 포트폴리오 자동 생성
- 카카오페이 / 토스 결제 연동
