# 엔로그 (Enlog)

> 한국 N잡러와 프리랜서를 위한 수입 관리 + 세금 계산 앱

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

---

## 서비스 소개

엔로그는 사업자 등록 없이 3.3% 원천징수로 일하는 N잡러와 프리랜서를 위한 올인원 수입 관리 앱입니다.

기존 앱(머니핀, 비즈넵)은 사업자 등록자 대상이라 55만 N잡러가 수입 관리를 카카오톡과 엑셀로 해결하고 있습니다. 엔로그가 이 문제를 해결합니다.

---

## 핵심 기능

- **수입 기록** — 클라이언트, 금액, 날짜 입력 / 실수령액 자동 계산 (3.3% 공제)
- **세금 계산** — 종합소득세 환급 예측 / 건강보험료 인상 경고
- **프로젝트 관리** — 클라이언트별 프로젝트 진행 상태 관리
- **간편 계약서** — 한국식 프리랜서 계약서 템플릿 + PDF 생성
